# Healthcare_Python_Projects
Healthcare Python Projects

Project 1: Hospital Patient Record System
-----------------------------------------

Description:
This project is a simple Python program used to store and manage patient records in a hospital.

Features:
- Add patient details
- View stored patient records
- Exit the system

Technologies Used:
- Python
- Lists
- Dictionaries
- Loops
- Conditional statements

How to Run:
1. Open the file hospital_record.py
2. Run the program using Python
3. Choose options from the menu to add or view patient records

Purpose:
This project helps beginners understand how to use Python for data storage and simple hospital management.


Project 2: Medicine Reminder Program
------------------------------------

Description:
This project is a simple Python program that reminds a user to take medicine at a specific time.

Features:
- User enters medicine name
- User enters reminder time
- Program displays reminder message

Technologies Used:
- Python
- Input and Output
- Variables

How to Run:
1. Open the file medicine_reminder.py
2. Run the program using Python
3. Enter the medicine name and time when prompted

Purpose:
This project demonstrates basic Python programming concepts like user input and message display.
